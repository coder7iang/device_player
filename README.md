## macOS 首次打开提示

由于应用未经 Apple 公证，从网络下载后 macOS 会拦截打开。把 `DevicePlayer.app` 拖到 `/Applications` 后，在终端执行一次以下命令即可正常使用：

```bash
sudo xattr -cr /Applications/DevicePlayer.app
```

之后双击打开即可，无需再每次去"系统设置 → 隐私与安全性"放行。
